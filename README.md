# Horologium: Enhanced

An Astral Sorcery addon that enhances the Horologium constellation ritual.

By default, Horologium randomly accelerates nearby machines.
This addon allows you to precisely choose which machines get accelerated.

---

# Features

## Ritual Adjuster Wand

* Right-click a machine to add it to the selection list
* Shift + Right-click ritual pedestal/anchor to bind machines
* Shift + Mouse Wheel to switch bind modes

---

## Binding System

* Mutual exclusion between pedestal and anchor
* Range-limited machine bindings
* Automatic disconnect when out of range
* Persistent saved binding data

---

## Visual Feedback

* Floating text overlays
* Ritual status display
* Constellation information
* Machine highlight rendering

---

## Advanced Support

* Alcara constellation freeze support
* Configurable overlay system
* Real-time ritual information

---

# TL;DR

Vanilla Horologium:

> Randomly accelerate nearby machines

Horologium: Enhanced:

> Accelerate only the machines you choose

---

# Dependencies

* Minecraft 1.12.2
* Forge
